# NEXUS - Premium AI Waitlist Landing Page

Une landing page ultra haut de gamme pour un produit IA futuriste, avec système de waitlist personnalisé, animations premium et design minimaliste sophistiqué.

## 🚀 Stack Technique

- **Framework**: Next.js 15+
- **Language**: TypeScript
- **Styling**: Tailwind CSS (fully customized)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Performance**: Optimized for production

## ✨ Caractéristiques

### 1. Hero Section Cinématique
- Animations fluides d'apparition
- Gradient backgrounds dynamiques
- Point focal dominant
- Parallax effect au scroll
- Scroll indicator animé

### 2. Système de Waitlist
- Validation email inline
- Confirmation personnalisée par IA
- Archétypes utilisateurs (Visionary, Builder, Insider, Pioneer, Innovator)
- Messages adaptés à l'heure du jour
- Animation de succès spectaculaire

### 3. Système de Parrainage
- Code de parrainage unique
- Copie vers presse-papiers
- Partage social
- Barre de progression
- Récompenses paliers

### 4. Compteur Live de Waitlist
- Animation count-up
- Simulation d'inscriptions en temps réel
- Indicateur pulsant
- Design minimaliste

### 5. Section Fonctionnalités
- 3 cartes premium
- Hover effects élégants
- Icônes animées
- Gradients subtils

### 6. Social Proof
- Testimonials authentiques
- Design minimaliste
- Animations de scroll reveal

### 7. Design Premium
- **Palette personnalisée**: Cyan, Purple, Slate
- **Typographie**: Space Grotesk (display) + Inter (body)
- **Glassmorphism** léger et contrôlé
- **Glow effects** subtils
- **Grain animé** en arrière-plan
- **Responsive** jusqu'à 375px
